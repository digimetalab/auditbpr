# Folder Data — BPR Audit Intelligence System

Place BPR financial files in this folder:

| File | Contents | Format |
|------|----------|--------|
| `neraca.xlsx` | 5-year financial position (balance sheet) | Excel / CSV |
| `laba_rugi.xlsx` | 5-year profit & loss statement | Excel / CSV |
| `aset_produktif.xlsx` | 5-year productive asset quality | Excel / CSV |
| `rasio.xlsx` | 5-year OJK financial ratios | Excel / CSV |

**Units:** Thousands of Rupiah (Ribuan Rupiah)
**Note:** Data files are not committed to git (see .gitignore)
