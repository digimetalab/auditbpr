## Benchmarking Usage Guide

```
When comparing the target BPR against industry benchmarks:

1. Compare against NATIONAL AVERAGES (above)
2. Compare against REGIONAL AVERAGES (search OJK data for the province)
3. Compare against SAME-SIZE BPRs (categorize by asset range)
4. Note the TREND direction (improving or deteriorating vs industry)

Web search queries for latest data:
  "statistik perbankan BPR OJK [YEAR]"
  "kinerja BPR [PROVINCE] [YEAR]"
  "OJK BPR data statistik bulanan"
  "SPI statistik perbankan Indonesia BPR"
```
